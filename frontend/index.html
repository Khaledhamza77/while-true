Two Ways Agents Think

Loop Engineering ReAct Pattern

Loop Engineering builds agents around a tight Reason → Act → Observe cycle where the model reasons about what to do, calls a tool, then observes the result — repeating until the task is complete. This compact feedback loop makes every decision traceable and keeps the agent grounded in real tool outputs rather than pure reasoning.

Autoresearch Research Pipeline

Autoresearch structures the agent as a Plan → Search → Synthesize → Evaluate pipeline suited for open-ended knowledge tasks. The agent first builds a research plan, gathers sources, synthesizes findings, then self-evaluates quality — looping back to search when gaps remain and stopping only when the answer meets the set criteria.
