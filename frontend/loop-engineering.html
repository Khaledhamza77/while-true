Loop Engineering

The ReAct pattern — Reason, Act, Observe, Repeat

The Graph

How It Works

1
Reason

The LLM receives the conversation history and decides: call a tool to gather more information, or answer directly.

2
Act

If a tool call is decided, the search tool fires against the Tavily API, returning titles, URLs, and snippets.

3
Observe

The tool result is appended to the message history and the iteration counter increments.

4
Loop

Control returns to Reason. The agent now has richer context. It loops until it's confident enough to answer or the iteration cap (10) is hit.

When to Use This

Good fit

Real-time Q&A, factual lookups, tasks where you don't know how many searches will be needed in advance.

Not ideal

Deep multi-topic research where you need structured synthesis across many sources — use Autoresearch for that.
